# Beautiful PPT Skill

## 0. 先说结论（执行原则）

这是一个**强制分步**技能，不允许跳步，不允许只给泛泛大纲。

若用户要求做 PPT，必须依次完成：
1) 内容提炼  
2) 提示词生成（背景图/元素图/艺术字）  
3) 图片生成（若工具可用）  
4) 页级排版方案  
5) 质检与交付

## 1. 硬性约束（必须遵守）

- 必须输出“步骤进度清单”，并在每一步完成后更新状态。
- 必须给出可直接复制的提示词，不能只写“建议生成图片”。
- 必须包含“艺术字”设计与提示词（至少 3 个标题词）。
- 必须明确哪些页突出关键结论、哪些页突出关键数据（大字号/高对比）。
- 若无法直接生成图，必须明确阻塞原因，并交付完整可执行提示词包。

禁止行为：
- 只给 PPT 大纲，不给提示词。
- 没有背景图方案就进入排版。
- 不做艺术字方案。

## 2. 输入解析

最低输入：
- 目标（汇报对象、场景）
- 材料（文档、数据、图片）
- 页数（默认 10 页）

若用户给了 `ppt/pptx` 文件：先抽取其主题、结构、关键数据，再重设计，不得原样照搬。

## 3. 对比驱动背景图（第一优先级）

每张背景图提示词必须至少包含 3 组对比：
- 明/暗
- 冷/暖
- 大/小
- 少/多
- 动/静（任选其一补足）

固定模板：

```text
[主体]，[场景]，[对比1]，[对比2]，[对比3]，
[镜头语言]，[质感词]，[构图留白位置]，
16:9, cinematic, ultra-detailed, no watermark, no text
```

## 4. 风格一致元素图（跨页统一）

必须先定义 Style DNA（并在所有图中复用）：
- 主色/辅色/背景色
- 光照方式
- 纹理质感
- 镜头语言
- 情绪关键词

必须产出：
- 背景图提示词 >= 3 条
- 元素图提示词 >= 6 条（同风格，可抠图）
- 艺术字提示词 >= 3 条

## 5. 内容提炼与重点放大

从材料中必须抽取：
- 核心主张 1 句
- 关键结论 3-5 条
- 关键数据 5-8 条（含单位和时间口径）

并输出“重点放大映射”：
- 第 X 页：放大数据/结论是什么
- 放大方式：超大数字 / 高对比色块 / 艺术字标签 / 留白聚焦

## 6. 艺术字流程（封面 + 重点页）

每次都执行：
1. 为封面标题生成艺术字提示词（主题匹配）
2. 为 2-3 个重点词生成艺术字提示词
3. 说明透明底要求（如非透明，需抠图后再上版）

艺术字提示词模板：

```text
为“[标题文字]”生成艺术字，风格[书法/刀锋/金属/科技涂鸦]，
笔画有力量感，识别度高，透明背景，无额外文字，无水印。
```

## 8. 质检清单（交付前逐条自查）

- 是否有明确“对比逻辑”而不是普通配图
- 是否给了完整提示词而不是描述性语言
- 是否有艺术字方案
- 是否定义并复用了统一 Style DNA
- 是否把关键数据做了大字号或强对比突出
