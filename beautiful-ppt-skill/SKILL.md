# Beautiful PPT Skill

## 0. 先说结论（执行原则）

这是一个**强制分步**技能，不允许跳步，不允许只给泛泛大纲。

若用户要求做 PPT，必须依次完成：
1) 内容提炼  
2) 提示词生成（背景图/元素图/艺术字）  
3) 图片生成（若工具可用）  
4) 页级排版方案  
5) 质检与交付

## 1. 硬性约束（必须遵守）

- 必须输出“步骤进度清单”，并在每一步完成后更新状态。
- 必须给出可直接复制的提示词，不能只写“建议生成图片”。
- 必须包含“艺术字”设计与提示词（至少 4 个标题词：封面1个+重点页3个）。
- 必须明确哪些页突出关键结论、哪些页突出关键数据（大字号/高对比）。
- 若无法直接生成图，必须明确阻塞原因，并交付完整可执行提示词包。
- 封面和章节过渡页背景必须达到“电影感”标准，不允许普通素材感。

禁止行为：
- 只给 PPT 大纲，不给提示词。
- 没有背景图方案就进入排版。
- 不做艺术字方案。
- 背景图没有镜头语言/光影层次/景深氛围。

## 2. 输入解析

最低输入：
- 目标（汇报对象、场景）
- 材料（文档、数据、图片）
- 页数（默认 10 页）

若用户给了 `ppt/pptx` 文件：先抽取其主题、结构、关键数据，再重设计，不得原样照搬。

## 3. 电影感背景图（第一优先级）

每张背景图提示词必须至少包含 3 组对比：
- 明/暗
- 冷/暖
- 大/小
- 少/多
- 动/静（任选其一补足）

并且必须同时包含以下电影语法字段（缺一不可）：
- 机位与镜头：`wide-angle / low-angle / telephoto / aerial / dolly-in` 至少 1 个
- 光影结构：`rim light + volumetric light + deep shadow` 至少 2 个
- 空气与层次：`mist/smoke/particles/depth haze` 至少 1 个
- 画幅与留白：明确标题留白区（左/右/上）与主视觉落点

固定模板（必须照抄填空）：

```text
[主体/动作]，[场景与时代感]，
对比: [明暗] + [冷暖] + [大小 or 疏密]，
camera: [机位+镜头运动]，
lighting: [主光/轮廓光/体积光/阴影]，
atmosphere: [烟雾/水汽/颗粒/景深层次]，
composition: [主体落点] + [标题留白位置]，
texture: premium film still, high dynamic range, ultra-detailed,
16:9, no watermark, no text
```

电影感负面词（每条背景图都要附带）：

```text
negative prompt: flat lighting, clipart, low contrast, oversaturated cartoon, posterized, blurry, watermark, random text, cheap stock photo look
```

## 4. 风格一致元素图（跨页统一）

必须先定义 Style DNA（并在所有图中复用）：
- 主色/辅色/背景色
- 光照方式
- 纹理质感
- 镜头语言
- 情绪关键词

必须产出：
- 背景图提示词 >= 3 条
- 元素图提示词 >= 6 条（同风格，可抠图）
- 艺术字提示词 >= 3 条

## 5. 内容提炼与重点放大

从材料中必须抽取：
- 核心主张 1 句
- 关键结论 3-5 条
- 关键数据 5-8 条（含单位和时间口径）

并输出“重点放大映射”：
- 第 X 页：放大数据/结论是什么
- 放大方式：超大数字 / 高对比色块 / 艺术字标签 / 留白聚焦

## 6. 艺术字流程（封面 + 重点页，强制执行）

每次都执行：
1. 为封面标题生成艺术字提示词（主题匹配）
2. 为 3 个重点词生成艺术字提示词（如“原动力/突破/增长”）
3. 先生成艺术字图片，再抠图，再叠加到 PPT（不得只用系统字体替代）
4. 说明透明底要求（如非透明，需抠图后再上版）

艺术字提示词模板：

```text
为“[标题文字]”生成艺术字，风格[书法飞白/刀锋硬朗/金属雕刻/科技霓虹]，
笔画有力量感，重心稳定，边缘干净，高识别度，
只输出该词本身，透明背景 PNG，无额外文字，无水印。
```

抠图执行要求（必须输出在结果里）：
- 若生成结果非透明底：执行背景移除，导出透明 PNG。
- 给出每个艺术字的建议摆放页码、大小比例、叠加模式（正常/滤色/叠加）。

## 8. 质检清单（交付前逐条自查）

- 是否有明确“对比逻辑”而不是普通配图
- 是否给了完整提示词而不是描述性语言
- 是否有艺术字方案且包含“生成->抠图->落版”
- 是否定义并复用了统一 Style DNA
- 是否把关键数据做了大字号或强对比突出
- 背景是否满足电影语法（机位+光影+空气层次+留白）
