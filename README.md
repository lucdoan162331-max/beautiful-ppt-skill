# beautiful-ppt-skill

A Cursor Agent Skill for building premium-looking PPT decks with:
- contrast-driven cinematic backgrounds,
- consistent cross-slide visual language,
- data-first storytelling,
- artistic title assets for high-impact covers and highlights.

## Repository Layout

```text
beautiful-ppt-skill/
└── SKILL.md
```

## Install (Personal Skill)

Copy the skill folder to your local Cursor skills directory:

```bash
mkdir -p ~/.cursor/skills
cp -R beautiful-ppt-skill ~/.cursor/skills/
```

After copy, expected path:

```text
~/.cursor/skills/beautiful-ppt-skill/SKILL.md
```

## How to Invoke

In Cursor chat, explicitly mention this skill and ask for PPT production. Example:

```text
Use beautiful-ppt-skill to build a 10-page deck from my materials.
Theme: marine tech.
Audience: executive team.
```

## What It Produces

- slide-by-slide structure,
- background/element/art-text prompt pack,
- key-number emphasis plan,
- final quality checklist.
